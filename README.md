# Filmorate
Simple service with REST API, works with films and user ratings, as well as return the top-5 recommended films to watch.

### Entities 
1. User
2. Film


## Application functions:
## Film
- Adding a film
- Updating a film
- Retrieving all films
- Like films
- Popular films list

## User
- Creating a user
- Updating a user
- Retrieving a list of all users
- Add friends
- Get common friends
